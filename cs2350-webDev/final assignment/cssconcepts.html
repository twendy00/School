CSS Box Model

The CSS Box Model Explained

All HTML elements can essentially be viewed as boxes, which is the box model. The box model allows us to size, space, position, and add any properties (ex. color, background, etc.) to HTML elements. These boxes consist of a margin, border, padding, and content.

  • Margin: the space around the border. It can create distance between elements on the page.
  • Border: the area or line that exists between the padding and margin.
  • Padding: the space between the content and border.
  • Content: the content of the element such as text, an image, video, etc.

The Padding Property

As stated above, the padding determines how much space is between the content and the border. The top, right, bottom and left spaces can be determined individually. In this box, the padding has been set to 30px all around, so the space around the content is equidistant on all sides.

The padding in this box has been set to 50px on the top and bottom and 125px on the left and right. You should be able to see how the top and bottom spaces between the content and border are more narrow than the left and right spaces.

In this box, the padding on each side has been set to a different size. The top is at 150px, the right is at 120px, the bottom is at 75px and the left is at 50px.You should be able to see how the space between the content and border are all different sizes.

The Margin Property

The margin property determines how much space exists beyond the border surrounding the box element. By adjusting the margin-top, margin-left, margin-bottom and margin-left properties, more space can be used to create space between any other box elements. In this example, the margin is set to default.

This box has the margins set to 50px on the left side. You should be able to see that it's been shifted out from the right by 50px.

This box has the margins set to 160px on the left side. You should be able to see that it's been shifted out from the right by 160px.

Positions

Relative Position

Relative positioning sets the position relative to itself. Changing the top, right, bottom and left will move the image from where it would naturally appear. The bee picture below has been set to a relative position, and it has been moved 10px from the left.

Absolute Position

Elements with an absolute position will stray outside of the document flow and anchor itself to its first positioned ancestor. If there are no positioned ancestors, then the element will anchor itself to the initial document html. Changing the top, right, bottom and left properties will cause the element to move relative to its parent element. For example, the iceberg picture below is set to absolute and at 200px from the right. It also has a z-index, which determines overlapping images. Since the z-index is positive at 2, it will be placed in the forefront above the fireworks image.

Fixed Position

Last but not least, the fireworks image to the right has a fixed position. A fixed position causes the element to stay anchored with the HTML document as a whole. Scrolling will not impact it, and it will remain on the page when scrolling. Moving the image through the top, right, bottom and left properties will cause it to move relative to the HTML document. In the fireworks example, the image is set to 500px from the bottom and 0px from the right. The z-index has been set to -2 so it will go behind any other elements that overlap it.

Shadows

Horizontal Property

Horizontal Property

The h-shadow is the first element of the text-shadow property. It specifies how horizontally far away the shadow should be from the text. The first h-shadow text above is set at 3px while the second v-shadow text is set to 10px. When using the text-shadow property, this is a required field.

Vertical Property

Vertical Property

The v-shadow is the second element of the text-shadow property. It specifies how vertically far away the shadow should be from the text. The first v-shadow text below is set at 3px while the second v-shadow text is set to 10px. When using the text-shadow property, this is a required field.

Blur Property

Blur Property

The blur-radius is the third element of the text-shadow property. Setting the blur value higher will cause the shadow to become wider and lighter. The first blur shadow is set at 3px, but the second blur value is set to 10px.
